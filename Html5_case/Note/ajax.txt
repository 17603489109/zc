1、ajax概念：AJAX即“Asynchronous Javascript And XML”（异步JavaScript和XML），
   是指一种创建交互式网页应用的网页开发技术。简单说就是：ajax就是一种数据交互技术。
2、ajax用途：客户端与服务器的数据交互。用的好的情况下，还可以为服务器分压。
3、异步：就是每个ajax请求过程中与网页加载进度无关，相互之间不影响。
   优势：在页面未加载完成之前用户依然可以流畅且自由操作页面的其他内容。
5、同步：是相对于异步而言的。也就是说同步和网页加载的必须是同时进行的。
   缺点：网速慢的时候会出现浏览器卡顿，甚至假死。导致用户不能自由操作浏览器。必须等待所有数据全部加载完成。
6、同源策略：同域名、同协议、同端口 称之为同源策略。缺一不可，否则称为非同源。

授课内容：
1、兼容问题（主要解决低版本的IE）

2、封装ajax（可以一次编写多次调用）

3、回掉函数（利用指针调用的函数叫回掉函数）
3、json（一个数据格式）：JSON(JavaScript Object Notation) 是一种轻量级的数据交换格式。它基于ECMAScript的一个子集。       JSON采用完全独立于语言的文本格式，但是也使用了类似于C语言家族的习惯（包括C、C++、C#、Java、JavaScript、Perl、Python等）。这些特性使JSON成为理想的数据交换语言。 易于人阅读和编写，同时也易于机器解析和生成(一般用于提升网络传输速率)。

4、jsonp（跨域请求数据的一个协议或者叫跨域访问一个模式）
   JSONP(JSON with Padding)是JSON的一种“使用模式”，可用于解决主流浏览器的跨域数据访问的问题。由于同源策略，一般来说位于 server1.example.com 的网页无法与不是 server1.example.com的服务器沟通，而 HTML 的<script> 元素是一个例外。利用 <script> 元素的这个开放策略，网页可以得到从其他来源动态产生的 JSON 资料，而这种使用模式就是所谓的 JSONP。用 JSONP 抓到的资料并不是 JSON，而是任意的JavaScript，用 JavaScript 直译器执行而不是用 JSON 解析器解析。

5、跨域：是指在非同源策略下进行的数据交互
应用场景：OA与CRM之间的数据交互、CRM与进销存系统的数据交互等。一般跨平台，跨系统，尤其是在一个企业中很容易遇到跨域的情况的。
